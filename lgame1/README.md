# L'Oréal Beauty Memory Match

Мобильная веб-игра в жанре "Memory Match" на русском языке, созданная для L'Oréal Paris.

## Демо

https://andrewkk875.github.io/my-app/lgame1/

## Описание

Игрок открывает карточки и находит пары культовых продуктов L'Oréal Paris. За победу начисляются очки и открываются достижения.

## Экраны

| Экран | Описание |
|-------|----------|
| Главный | Логотип, продукты, кнопка "Играть" |
| Онбординг | Инструкция как играть |
| Лобби | Статистика, выбор режима |
| Выбор уровня | Лёгкий (6 пар), Средний (12), Сложный (20) |
| Игра | Сетка карточек с таймером |
| Победа | Счёт, звёзды, переход дальше |
| Поражение | Статистика, повторная попытка |
| Продукт | Карточка продукта с ценой |
| Glow Challenge | Ограниченное событие |
| Профиль | Достижения, статистика |
| Рейтинг | Таблица лидеров |
| Ежедневная награда | Награды за 7 дней подряд |

## Уровни сложности

- **Лёгкий** — 6 пар, 60 секунд
- **Средний** — 12 пар, 90 секунд
- **Сложный** — 20 пар, 120 секунд

## Технологии

- Чистый HTML / CSS / JavaScript (без фреймворков)
- Один файл `index.html` (~2000 строк)
- GitHub Pages для деплоя

## Структура файла

```
index.html
├── <style>   — весь CSS (~950 строк)
└── <script>  — вся логика (~300 строк)
    ├── allCards[]       — 20 продуктов L'Oréal
    ├── LEVEL_CONFIG     — настройки уровней
    ├── TIMER_CONFIG     — время на уровень
    ├── initGame()       — создание карточек
    ├── onCardClick()    — логика переворота
    ├── startTimer()     — обратный отсчёт
    └── showScreen()     — навигация между экранами
```

## Продукты в игре

20 продуктов брендов: L'Oréal Paris, YSL, Kérastase, Redken, La Roche-Posay, Vichy, Maybelline, Biotherm, IT Cosmetics.

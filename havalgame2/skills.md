# Havalgame2 — Three.js Crossy Road Clone

## Концепция
- Жанр: endless runner / crossing
- Движок: Three.js
- Цель: мобильная игра, стабильные 60fps

## Источники и референсы

- **Гист с игрой:** https://gist.github.com/Cayey514/264e4d4db8fd3be1108fe861adc74b8e
- **YouTube туториал:** https://youtu.be/vNr3_hQ3Bws

## Анализ референса (Gist)

**Механики:**
- Ортографическая камера, следит за игроком
- Процедурная бесконечная генерация рядов (лес + дорога)
- Типы рядов: лес (деревья-препятствия), дорога (машины), дорога (грузовики)
- Игрок: куб-тело + куб-шапка, прыжки по клеткам с синусоидальной анимацией
- Очередь ходов для плавного управления
- Машины/грузовики с разными скоростями (125, 156, 188 px/s), рандомное направление
- Коллизия через bounding box
- Game Over при столкновении с транспортом

**Структура:**
- index.html + script.js + style.css
- Модули: Camera, Car, Truck, Tree, Player, Road, Grass
- Шрифт: Press Start 2P

**Освещение:**
- DirectionalLight + тени (2048×2048 shadow map)
- AmbientLight
- Материалы: Lambert + Phong

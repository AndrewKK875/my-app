# Как работаем

1. **Сначала план** — перед любым изменением файлов предлагаю, что именно собираюсь сделать.
2. **Жду подтверждения** — не приступаю к изменениям, пока не получу согласие.
3. **Комментирую шаги** — коротко сообщаю, что делаю на каждом этапе.
4. Мелкие правки (опечатка, одна строка) — можно без плана, просто делаем.

# Частые задачи

- **Анализ таблицы** → возвращать `.md`-файл с выводами (структура: общая картина → тренды → рекомендации).
- **Финансовые задачи** → цены всегда в двух валютах: ₽ и € (курс указывать явно).
- **Excel-задачи** → использовать Node.js + `exceljs` или `xlsx`; Python не установлен.
- **3D / игры** → стек: Three.js, GLB-модели, DRACOLoader, vanilla JavaScript.
- **Графики в Excel** → рендерить через `chartjs-node-canvas`, вставлять как PNG-изображение.

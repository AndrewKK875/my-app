# Havalgame3 GitHub references

The video reference is a 3D low-poly arena shooter / tank shooter with UFO enemies. These repositories are for studying mechanics and implementation patterns only; Havalgame3 should be a new original project.

## Closest mechanics

1. https://github.com/mrdoob/three.js/tree/dev/examples/games/fps
   - Official Three.js FPS example. Useful for basic 3D movement, projectiles, collisions, and scene structure.

2. https://github.com/jeromeetienne/threex.tankgame
   - Three.js tank game experiment. Useful for tank-style movement and turret/game object structure.

3. https://github.com/yandeu/tankgame
   - Browser tank game. Useful for top-down tank movement, bullets, and combat loop ideas.

4. https://github.com/agar3s/space-shooter
   - Three.js space shooter. Useful for enemy waves, bullets, score, and simple arcade pacing.

5. https://github.com/picosonic/js13k-2021
   - Small JavaScript shooter-style game. Useful for compact architecture and lightweight effects.

## Arena / shooter architecture

6. https://github.com/schteppe/cannon.js/tree/master/demos
   - Physics demos for Cannon.js. Useful if the arena needs vehicle collisions or physical projectiles.

7. https://github.com/pmndrs/react-three-rapier
   - Rapier physics bindings for Three.js/React Three Fiber. Useful reference for modern collision and rigid bodies.

8. https://github.com/pmndrs/react-three-fiber
   - React renderer for Three.js. Useful if Havalgame3 becomes a React-based game later.

9. https://github.com/lo-th/phy
   - JavaScript physics playground/examples. Useful for vehicle, raycast, and collision references.

10. https://github.com/mrdoob/three.js/tree/dev/examples
    - Three.js examples collection. Useful for GLB loading, shadows, lights, postprocessing, sprites, particles, and controls.
